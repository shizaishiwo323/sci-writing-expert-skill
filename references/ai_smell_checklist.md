# AI-Smell Checklist

Use this checklist for expression only. Do not use it to change scientific logic or content.

## High-priority signals

| Signal | Typical pattern | Preferred action |
|---|---|---|
| undefined fancy term | `mechanism-diagnostic bridge` | define or use a plain relation |
| decorative intensifier | `remarkably`, `fundamentally` | delete or quantify |
| overstrong claim | `proves`, `ensures` | calibrate to evidence |
| empty contribution phrase | `provides new insights` | state the concrete finding |
| long abstract subject | `The establishment of a comprehensive...` | move the real actor to subject |
| heavy nominalization | `the occurrence of` | use a direct noun or verb |
| rare decorative verb | `unveil`, `elucidate` | use a precise ordinary verb |
| invented compound | `connectivity-breakthrough nexus` | rewrite as ordinary syntax |
| unclear causal jump | observation followed by settled mechanism | separate and calibrate |
| scope loss | removing `in simulations` | restore scope |

## Diagnostic labels

- `undefined-term`
- `overclaim`
- `decorative-intensifier`
- `unnecessary-compound`
- `heavy-nominalization`
- `rare-word`
- `long-subject`
- `unclear-causal-link`
- `scope-drift`
- `technical-content-risk`

## Forbidden fixes

Do not:

- add a mechanism, citation, or literature judgment;
- create a new named framework;
- remove uncertainty markers;
- remove numbers, units, variables, citations, or figure labels;
- turn a local sentence edit into an unrequested paragraph rewrite;
- replace a precise technical term with a vague simple word.

## Quick pass

1. Is the issue expression rather than science?
2. Can a local edit fix it?
3. Is evidence strength unchanged?
4. Are protected elements intact?
5. Did the revision create a decorative label?
