# Cohesion and Paragraphs

## 1. Paragraph function

A scientific paragraph normally has one dominant function:

- establish context;
- synthesize a literature theme;
- define a gap;
- justify a method;
- describe one procedure;
- report one result pattern;
- compare regimes;
- interpret a mechanism;
- discuss one limitation;
- derive one implication.

Several moves may appear, but they should support one main point.

## 2. Topic sentence

A useful topic sentence:

- identifies the subject;
- links to the preceding paragraph;
- predicts the evidence or reasoning that follows;
- does not promise more than the paragraph delivers.

Do not force a generic topic sentence into a tightly procedural Methods paragraph where sequence already organizes the content.

## 3. Evidence sequences

Common patterns:

```text
claim -> evidence -> explanation -> bounded implication
general pattern -> details -> exception -> synthesis
method choice -> implementation -> quality control -> limitation
prior approach A -> prior approach B -> unresolved issue
observation -> comparison -> mechanism -> consequence
```

Reorder only when the current sequence blocks understanding.

## 4. Sentence-to-sentence cohesion

Use:

### Controlled lexical overlap

Repeat a key scientific noun when repetition prevents ambiguity. Scientific writing does not require a synonym for every occurrence.

### Explicit pro-forms

Use `this result`, `these measurements`, or `this approach` only when the referent is unmistakable. Avoid bare `this` when it could refer to several things.

### Old-to-new flow

Begin with active/known information and end with the new point that the next sentence develops.

### Accurate relation

Use a connector only when it names the real relation.

### Structural parallelism

Parallel form can signal comparison, sequence, or contrast.

## 5. Paragraph boundaries

Start a new paragraph when:

- the dominant function changes;
- a new result pattern or mechanism begins;
- the discourse moves from observation to a substantial interpretation;
- focus changes to another method, condition, population, or scale;
- a limitation or implication needs independent development.

Avoid both fragmented one-sentence paragraphs and long paragraphs with several unrelated jobs.

## 6. Paragraph ending

A paragraph may end with:

- synthesis;
- local implication;
- limitation;
- forward pointer;
- the concept taken up next.

Do not end every paragraph with an exaggerated importance claim.

## 7. Revision method

1. Label each sentence by function.
2. State the paragraph's main job in one line.
3. Identify off-function sentences.
4. Reorder by a defensible evidence sequence.
5. Add only necessary transitions.
6. Revise sentence form after order is stable.
7. Check the exit to the next paragraph.
