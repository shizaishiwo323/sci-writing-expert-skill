# Plain Word Choice

## Rule

Prefer a simpler word only when it preserves scientific meaning, evidence strength, technical relation, scope, and uncertainty.

If replacement may change meaning, keep the original or mark `author-confirm`.

## Common opportunities

| Question | Often clearer |
|---|---|
| utilize | use |
| leverage | use |
| elucidate | show, explain |
| delineate | define, separate, distinguish |
| facilitate | support, help, allow |
| commence | begin |
| terminate | end |
| necessitate | require |
| possess | have |
| prior to | before |
| subsequent to | after |
| in order to | to |
| due to the fact that | because |
| with regard to | about, for |
| is capable of | can |
| is indicative of | indicates |
| shed light on | show, help explain |
| offer insights into | state the specific finding |

## Nominalization reduction

| Wordy form | Direct form |
|---|---|
| the occurrence of dissolution | dissolution |
| the development of cracks | cracks develop |
| the expansion of the channel | the channel expands |
| the reduction of tortuosity | tortuosity decreases |
| the enhancement of permeability | permeability increases |
| the implementation of the method | implementing/using the method |

## Do not oversimplify

Keep field-standard terms such as `permeability`, `porosity`, `tortuosity`, `advection`, `diffusion`, `inversion`, or any precise domain term.

Do not replace an association with a causal verb merely because the causal verb is shorter.
